Developer Communities

The Open Geospatial Consortium (OGC) is an international consortium of more than 500 businesses, government agencies, research organizations, and universities driven to make geospatial (location) information and services FAIR - Findable, Accessible, Interoperable, and Reusable. We have a very active developer community, so there is always new insight to discover and new knowledge to share. Keep up with What's New through this website.

  • OGC Web Services

    Join the community of OGC Web Services (OWS) developers.

  • User Interface

    Join the community of User Interface (UI) developers.

  • OGC APIs

    Join the community of OGC API developers.

  • Encodings

    Join the community of developers of reader and writer software.

  • Executable Test Suites

    Join the community of developers of Executable Test Suites.

  • Open Source

    Join the many Open Source communities that develop geospatial software that implements OGC standards.

  • OGC Members

    Join the developer communities of OGC Member organizations.

  • Support

    If you have a question, you can find answers here. OGC staff and members are always available to help.
